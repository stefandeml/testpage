Let’s jump into ZoKrates by working through a hands-on project together! This chapter introduces you to the basic features of ZoKrates and to how to use zkSNARKs in a real-world example.

We’ll implement a problem that's very typical in blockchain use-cases: proving the knowledge of a pre-image for a given hash digest. Here’s how it works: //TODO: End2end example of Ethereum SHA256 including solidity and python example

This example creates the equivalent output as the following solidity code:

pragma solidity ^0.4.24;
contract SHA256Test {
        event Success(
        bytes32 indexed _id
    );

    function calc_sha() public returns (bytes32)  {
        bytes32 a = 0x0;
        bytes32 b = 0x5;

        bytes32 result = sha256(a,b);
        emit Success(result);
        return result;
    }
}
